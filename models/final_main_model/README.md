# Final Main Model

This directory contains only the final selected main model artifact.

Included files:

- `task_04_multimodal_mlp_fusion.pt`: final multimodal MLP fusion model.
- `task_04_multimodal_mlp_metrics.json`: metrics for the final multimodal model.

All other model artifacts, including AutoGluon `.pkl` files and CT ResNet `.pth` weights, are intentionally excluded from this GitHub package. Their training scripts, parameters, metrics, prediction files, and reproduction notes are retained so users can retrain them if needed.
